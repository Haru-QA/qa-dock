# QA Dock — QA 팀 대시보드

QA팀 전체 현황을 한눈에 파악할 수 있는 통합 대시보드입니다.

## 주요 기능

1. **QA 상황판** — QA 필요 / QA 리뷰 / QA 진행 / QA 완료 4단계 현황 보드
2. **QA 워크플로우** — 현재 프로세스 단계를 상단에 간략 표시
3. **통계/분석** — 상태별, 우선순위별, 담당자별, 기간별 차트
4. **도구 연동** — 정적분석(D), TC 설계(E), CI/CD(C) 결과 통합 표시
