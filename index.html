Bug Priority

Blocker앱 사용 불가 (Crash, Login Fail)
Critical심각한 기능 문제 (결제, 방송 오류)
Major눈에 띄는 문제 (기대 결과와 다름)
Minor사소한 문제 (텍스트, 낮은 재현율)
Trivial거의 문제 없음 (유사 표현 차이)
